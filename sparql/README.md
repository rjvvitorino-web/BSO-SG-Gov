# SPARQL Queries

SPARQL queries used against the GraphDB triple store are maintained
in a private repository. Documented queries include:

- Strategic gap detection — identifies objectives without measurable indicators
- Causal network validation — verifies cause-effect relation completeness
- Indicator coverage — measures measurement system completeness across perspectives
