# Ontology Files

The core BSO ontology (OWL/RDF) and SHACL validation shapes are maintained
in a private repository due to attribution and confidentiality obligations.

These files include:
- `BSO_SGPCM.rdf` — Core Balanced Scorecard Ontology defining Strategic Objectives,
  Perspectives, Indicators, and cause-effect relationships
- `shacl_shapes.ttl` — SHACL validation shapes enforcing ontological constraints
  (e.g. every Strategic Objective must have at least one Indicator)
